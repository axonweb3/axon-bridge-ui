# axon-bridge-ui

Frontend UI for the Axon Bridge. 

### How to run

```
yarn
yarn start
```

### How to build

```
yarn
yarn build
```

### Add new environment variables

Open `.env` file, and add lines like this:

```
REACT_APP_XXX_ZZ="1234xdfrd"
REACT_APP_YYY_AA="sdfasdfas"
```
